# Luxury Travel Consultant AI

## Overview

This project demonstrates advanced system prompting and persona engineering techniques using a luxury travel assistant AI.

The assistant maintains a strict professional persona, follows behavioral constraints, handles difficult customer interactions, and resists prompt injection attacks.

---

## Features

- Luxury travel consultant persona
- Strict behavioral constraints
- Few-shot prompting examples
- Jailbreak resistance testing
- Controlled response formatting
- Hallucination prevention

---

## Technologies Used

- ChatGPT / GPT-4
- Prompt Engineering
- Few-Shot Prompting
- Persona Design

---

## Files

- `system_prompt.md` → Main system instructions
- `few_shot_examples.md` → Example ideal conversations
- `jailbreak_tests.md` → Safety and prompt attack testing

---

## Key Learning Outcomes

- Designing structured AI personas
- Applying prompt engineering techniques
- Preventing prompt injection attacks
- Building safer AI interactions
