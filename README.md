# DecodeLabs Project 1

This repository contains all tasks completed during Project 1 of the DecodeLabs Generative AI Internship.

The project focused on Prompt Engineering, RAG concepts, multimodal AI workflows, image generation, and AI safety testing.

---

# Tasks

## Task 1 — System Prompt Architect

Built a luxury travel assistant prompt system with:
- strict persona behavior
- tone constraints
- few-shot prompting
- hallucination control

---

## Task 2 — Creative Visionary

Generated Cyberpunk-Corporate themed AI visuals using prompt engineering techniques.

Included:
- logo concepts
- futuristic dashboard designs
- cinematic visual prompts
- negative prompting

---

## Task 3 — Context & RAG Workflow

Created a basic Retrieval-Augmented Generation (RAG) workflow using:
- PDF ingestion
- chunking
- embeddings
- vector database
- semantic search
- citation-based responses

---

## Task 4 — Multimodal Content Engine

Built a workflow that:
- transcribes video/audio using Whisper
- detects viral segments
- generates captions
- suggests AI B-roll ideas

---

## Task 5 — AI Safety & Bias Audit

Performed:
- jailbreak testing
- hallucination checks
- bias testing
- safety framework documentation
